Donor Database Metadata Repair

One-time tool to add missing year/month helper fields for faster Tracking and Statistics loading.

IMPORTANT: Download a backup first before running repair. This tool does not delete donor records and does not change names, serial numbers, blood types, or donation details. It only adds or fixes helper fields: collectionYear, collectionMonth, collectionMonthName, and collectionMonthKey.
Use Repair Missing Fields Only first. Use Repair Missing + Wrong Fields only if the scan says some records have wrong month/year metadata.
