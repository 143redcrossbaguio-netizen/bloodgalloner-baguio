How date carry-forward works:
• If a row has a date (e.g., March 1), that date is saved as the "last date".
• The next rows with empty date cells will automatically inherit March 1.
• When a new date appears (e.g., March 2), that becomes the new "last date".
• This repeats for the entire sheet — dates cascade forward until a new one is found.
• Rows with inherited dates are marked YES in the "Inherited Date?" column.

File support: Excel (.xlsx, .xls), CSV, and JSON (array of objects or named sheets).
JSON format: Array of objects, { "Sheet1": [...] }, or SheetJS workbook.
